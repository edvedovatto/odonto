# Regras de interface

## Direção visual

1. Visual de clínica, limpo e confiável.
2. Fundo claro.
3. Azul como cor principal.
4. Cinza claro para bordas e fundos.
5. Verde só para pago ou concluído.
6. Vermelho só para cancelado.
7. Logo provisório em quadrado azul simples.

## Regras de UX

1. Tela principal sempre é a agenda do dia.
2. Um foco principal por tela.
3. Botões grandes.
4. Poucos campos por formulário.
5. Fluxos curtos.
6. Sempre mostrar data, hora, paciente, médico e status de forma clara.
7. CPF sempre mascarado em listas.
8. Foto do paciente pequena e opcional.
9. Estados vazios claros.
10. Nada de menu profundo desnecessário.

## Comportamento mobile

1. Priorizar toque com o polegar.
2. Evitar modais enormes.
3. Evitar teclado surgindo o tempo todo.
4. Formulários curtos e com auto complete quando fizer sentido.
5. Tudo precisa parecer app instalado, não site genérico.

## Tom do texto

1. Simples.
2. Direto.
3. Sem jargão.
4. Sem telas explicando demais.
